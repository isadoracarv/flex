O que é display flex?

É uma propriedade CSS fundamental para criar layouts responsivos e alinhamentos complexos de forma simples, organizando elementos filhos dentro de um contêiner pai.

O que é justify-content?

É uma propriedade CSS usada para alinhar itens ao longo do eixo principal de um contêiner.

O que é align-itens?

É sada para posicionar elementos verticalmente ou horizontalmente, controlando o alinhamento de todos os itens.
